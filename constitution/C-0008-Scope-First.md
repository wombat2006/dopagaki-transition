# C-0008 Scope First

## Principle

説明範囲を超える主張をしてはならない。

## Articles

理論は説明可能な範囲を明示しなければならない。

未検証の領域は未検証として扱う。

推測と採択済み知識を混同してはならない。

理論の適用範囲外における主張は、仮説として扱う。

説明能力は理論の価値を高めるが、説明能力を超える一般化は理論を弱める。

AS-IS と TO-BE を明確に区別しなければならない。

採択済み知識と将来構想を同一の確実性で扱ってはならない。
