# The Dopagaki Transition

**From Dopamine Economy to Meaning Economy: A Neuropsychiatric, Socioeconomic and Civilizational Perspective**

This repository manages the manuscript, review history, and conceptual notes for *The Dopagaki Transition*.

## Core thesis

Dopagaki is not treated as a psychiatric disorder or moral failure. It is treated as a cognitive adaptation to an information-saturated environment shaped by dopamine-driven exploration, algorithmic reward systems, attention capitalism, and eventually meaning allocation systems.

## Key committed concepts

- Dopagaki is an adaptive cognitive state, not a disorder.
- Dopamine is treated primarily as exploration, expectation, and reward prediction, not merely pleasure.
- SNS functions as an industrial-scale variable-reward environment.
- Attention is finite, bounded by population, time, and cognitive bandwidth.
- Meaning is defined as an attention allocation mechanism.
- Religion will be treated as a meaning and attention allocation system.
