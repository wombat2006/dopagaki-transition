## この研究で表したいもの

インターネット上で「ドパガキ」と呼ばれる人たち——ショート動画を止められず、通知が来るたびに画面を開き、次の刺激を待ち続ける人たち——は、たいていこう説明される。

意志が弱い。若者の問題。SNS 依存。ドーパミン中毒。

本プロジェクトは、その説明をそのまま受け入れない。

**人間の脳は壊れていない。** 20 万年前のサバンナで「次に何があるか」を探す回路は、今も正常に動いている。変わったのは環境だけだ。探索の対象が、木の実から TikTok になっただけだ。

そして注意経済のなかでは、探索し続ける人間こそが **模範生** になる。工業社会の常識から見れば問題児だが、プラットフォームから見れば理想的な経済主体だ——この **大転倒（Great Inversion）** が、本研究の核心である。

```text
注意          … 21 世紀資本主義が工業化した資源
注意の有限性  … Attention ≠ ∞（母集団で上限が決まる）
意味の配分    … 有限の注意を、何に向けるか
宗教          … 注意配分の社会制度
文明          … 工業 → 注意 → 意味 → AI
```
