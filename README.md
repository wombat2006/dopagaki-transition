# ドパガキ・トランジション

## 副題

**ドーパミン経済から意味経済へ**

神経科学・行動心理学・政治経済学・宗教学・AI文明論から見た、高度情報社会の認知適応に関する試論。

---

## 概要

本プロジェクトは、インターネットスラングである「ドパガキ」を出発点として、ドーパミン、探索行動、SNS、Attention Economy、資本論、AI、宗教、文明遷移を横断的に分析する研究ノート兼論文プロジェクトです。

本論文では、ドパガキを単なる依存症や人格的欠陥としてではなく、**情報過剰環境において探索回路が適応した結果として生じる認知状態**として捉えます。
