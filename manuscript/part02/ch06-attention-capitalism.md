## 6.1 注意の工業化

20世紀資本主義は労働力を工業化した。

21世紀資本主義は注意力を工業化した。

```text
Industrial Society:  Value = Labor
Attention Society:   Value = Attention
```

産業革命は筋力を工業化した。工場、組立ライン、時間給——人間の身体能力を抽出可能な資源へ変換した。Industrial Capitalism は Labor Extraction を通じて Value Creation を行った。

情報革命は別の資源を工業化した。注意である。

Attention Capitalism は Attention Capture を通じて Value Creation を行う。

企業収益はもはや製品販売だけによって決定されない。人間の滞在時間、画面占有率、エンゲージメント——注意の量と質——が収益関数に組み込まれている。

したがって市場競争は必然的に、

> いかにして人間の報酬系を効率良く刺激するか

ではなく、

> いかにして人間の注意を長く拘束するか

へ収束する。

この現象はしばしば陰謀論的に語られる。しかし実際には単なる市場原理である。もしある SNS がユーザーの滞在時間を 1% 向上させれば、その企業は競争上優位に立つ。もしある動画プラットフォームが「次の動画」の精度を 0.5% 改善できれば、数十億ドルの価値差が生まれる。

結果として、世界最高峰のエンジニア群の相当部分は、核融合や宇宙開発ではなく、

> 「次の動画を見せる最適タイミング」

の研究に従事することとなった。

人類文明の到達点としてはなかなか味わい深い。

## 6.2 注意市場の形成

Attention as primary asset.

現代プラットフォーム企業の貸借対照表に載る最重要資産は、工場でも特許でもない。ユーザーの注意である。

```text
Industrial Capitalism
↓ Labor Extraction
↓ Value Creation

Attention Capitalism
↓ Attention Capture
↓ Value Creation
```

注意市場は以下の条件で形成された。

**常時接続性。** スマートフォンは注意を市場から切り離さない。睡眠中も通知は届く。

**アルゴリズム推薦。** 探索コストをゼロに近づける。ユーザーは「次を探す」必要がなく、「次が来る」のを待つだけになる。

**ネットワーク効果。** 注意が集まるほど、さらに注意が集まる。勝者総取りの構造。

**行動データの蓄積。** 注意は消費されるだけでなく、記録される。記録された行動は予測可能な資産へ変換される（Ch 10 へ）。

AI 革命は知能を工業化している。SNS 革命は注意力を工業化した。両者は独立ではない。推薦アルゴリズム、生成コンテンツ、パーソナライズドフィード——注意の工業化と知能の工業化は融合している。

ドパガキはこの注意市場の中で生きている。彼らは注意資本主義の理想被験者ではない——少なくとも Ch 11 までその議論は保留する——が、注意捕捉メカニズムに最も適応した行動様式を示している。

## 6.3 Part II への接続

Attention Capitalism の成立は、20世紀経済学の語彙だけでは説明できない。

労働価値説は「注意」を説明変数として持たない。資本主義/共産主義の二項対立も、注意市場を想定していない。

Ch 7 へ: なぜ資本論は obsolete なのか。なぜ通過しなければならないのか。

Ch 9 へ: Attention Scarcity——注意が有限であることの意味。

Attention Capitalism は、ドーパミン経済の完成形である。探索の工業化（Ch 4）、スキナー箱の planetary scale 化（Ch 5）を経て、注意が主要資源として市場に登場した。
